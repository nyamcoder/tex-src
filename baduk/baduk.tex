A: installing igo-stuff into the TeX-system (as a user)

0. identify $TEXMFHOME
$ kpsewhich -expand-var "\$TEXMFHOME"
/home/honggildong/texmf

1. creation of necessary folders according to the TDS:

$ mkdir -p ~/texmf/doc/latex/igo && mkdir -p ~/texmf/fonts/source/igo && mkdir -p ~/texmf/tex/latex/igo

2. download http://mirror.ctan.org/fonts/igo.zip, unzip, and copy -->

igo.sty --> ~/texmf/tex/latex/igo
igo/fonts --> ~/texmf/fonts/source/igo
rest -->  ~/texmf/doc/latex/igo

3. introducing all igo-stuff to the TeX-system (as a user):
$ texhash (~/texmf)



B: compiling the main document:
$ xelatex baduk(.tex)



~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

note I: metafont will be activated at the first run

note II: this xelatex-example uses ttf, for latex, pre-built type1-fonts can be fetched from here:  http://faq.ktug.org/faq/Karnes/2006-05?action=download&value=igo-type1-1.zip

note III: see other badul/igo-examples here: http://faq.ktug.org/faq/IgoPackage#s-3
